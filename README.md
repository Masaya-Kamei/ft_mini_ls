# 42 ft_mini_ls

Same result as this command `ls -1tr`.  
(No commandline arguments)

`./ft_mini_ls`

## Usage example

```zsh
git clone https://github.com/Masaya-Kamei/ft_mini_ls.git;
cd ft_mini_ls;
make;
echo -e "\n---Output---";
(cd unit_tests/testdir && ../../ft_mini_ls);
```
